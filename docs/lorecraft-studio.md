# Lorecraft Studio

Lorecraft Studio is the user-facing identity for this project. It is an agent-friendly creative studio for cinematic interactive game stories, not primarily a game engine.

## Positioning

Lorecraft Studio helps creators and agents produce:

- story bibles
- cinematic scenes
- characters
- dialogue
- branching narrative
- character emotions
- character stances
- scene staging
- background and image prompts
- lightweight animations
- gameplay hook specs
- playable previews
- export packages

## Flagship Project

The Last Testimony is the flagship project and should be treated as the main showcase. Secondary demos can remain useful examples, but they should not distract from the Chapter 1 polish pass.

The current Chapter 1 direction is a noir courtroom mystery with testimony navigation, evidence presentation, contradiction feedback, and cinematic payoff moments.

## Compatibility Names

Internal package names still use `@adventurekit/*` for now. This keeps the current workspace stable while the product identity transitions. Public UI, README copy, docs, browser metadata, and flagship content should use Lorecraft Studio.

## Addon Direction

Future systems should be presented as addons or modules:

- music studio
- image generation integration
- character art generation
- gameplay modules
- visual editor upgrades
- native/mobile packaging
- additional export targets
