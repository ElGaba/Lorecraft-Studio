# Export Format

Studio export packages are generated by `createExportPackage(game)` from `@adventurekit/core`.

## Files

- `game.json`: playable shell without Studio-only arrays.
- `story-bible.md`: premise, tone, visual direction, rules, cast, locations, and module plan.
- `chapter-outline.md`: story beats, source scene flow, and outcomes.
- `characters.json`: character records and variants.
- `scenes.json`: full scene records.
- `gameplay-hooks.json`: first-class hook specs.
- `gameplay-sequences.json`: implementation-ready hook sequence specs with linked scene ids, inputs, targets, effects, layout notes, and agent prompts.
- `assets.json`: asset prompt catalog.
- `evidence.json`: evidence-only Court Record items for inspection, contradiction, and chain-of-custody work.
- `variables.json`: initial serializable story variables.
- `animation-presets.json`: animation preset usage collected from scene blocks and character variants.
- `prompts.md`: scene, asset, and character prompt book.
- `implementation-plan.md`: runtime, hook module, and asset implementation notes.

## Use

The export panel in Studio lists the generated files and previews their contents. The package is deterministic for the selected in-memory project and can be regenerated after local edits.
